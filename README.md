# QuickEng-FE ✏️
: 숏폼 영상 기반 영어 학습 앱

### 1. 프로젝트 개요 및 기획 의도

- **개발 배경:** 현대인의 '숏폼 중독' 현상을 단순히 비판하는 것을 넘어, 공학적 관점에서 이를 '학습의 재료'로 역이용하고자 함.
- **기획 의도:** 무의미하게 소비되는 유튜브 쇼츠(Shorts) 영상에서 **Gemini AI**를 통해 핵심 영어 표현과 문맥(Context)을 추출하고, 이를 **로컬 DB**에 저장하여 사용자가 학습 카드로 활용할 수 있는 안드로이드 애플리케이션 개발.

### 2. 주요 기능

- **추천 숏츠**: 최신 트렌디한 숏폼 영상을 추천하여 영어 학습을 시작할 수 있습니다.
- **URL 등록**: 유튜브 URL을 입력하면, 자동으로 해당 영상의 학습 카드가 생성됩니다.
- **학습 카드**: 각 영상에서 학습할 핵심 문장들을 카드 형태로 제공하여 효율적인 학습을 지원합니다.
- **추가/삭제 기능**: 학습한 문장을 카드로 저장하고, 필요할 때 삭제하여 나만의 단어장을 관리할 수 있습니다.
- **트래커**: 학습 진도를 시각적으로 확인하고, 학습을 지속할 수 있는 동기 부여를 제공합니다.
